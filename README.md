# MyCar Rust

Car management desktop application built with Tauri (Rust + JavaScript).

## Features
- Vehicle management with car database
- Expense tracking with charts
- Service station directory
- Interactive map (OpenStreetMap)
- Technical inspections tracking
- OBD data logging
- Driver profile
- Statistics dashboard
- 12 languages (EN, RU, DE, IT, FR, ZH, DA, EL, ES, KO, PT, PL)
- Configurable font sizes
- Map position memory
